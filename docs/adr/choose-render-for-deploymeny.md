# Title
**Choosing Render for application deployment**

## Context
- Needed a platform to deploy our application with ease

## Decision
- Render provides a user-friendly interface for deploying applications, making it easier for developers to manage deployments without deep DevOps knowledge.
- Render offers automatic scaling capabilities, which is beneficial for handling varying loads without manual intervention.
- Render offers a Free tier for small projects, which is cost-effective for initial development and testing.
- Render supports multiple programming languages and frameworks, making it versatile for different project needs.
- Render provides built-in CI/CD capabilities, allowing for seamless integration and deployment workflows like github Actions.

## Consequences
It will help with deploying the application for testing in a cost effective way.
